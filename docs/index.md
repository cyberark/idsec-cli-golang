# Idsec CLI

![Idsec CLI Golang](./media/sdk.png){: style="height:80%;width:70%"}

The official CLI for CyberArk's Identity Security Platform Services.

## Motivation

Idsec CLI provides a command-line interface for general purpose automation of Identity Security Platform operations. Use it to configure profiles, authenticate, and execute actions across SIA, pCloud, Identity, Secrets Hub, and other services.
